## What is Mino?

Mino is an **AI-powered web automation API** that lets you automate any website using natural language. Instead of writing CSS selectors or XPath, you describe tasks in plain English.

- Works on any website (including authenticated, JS-heavy, bot-protected sites)
- Returns structured JSON results
- Real-time streaming via SSE

## Under the hood

Mino uses a real browser controlled by AI. You send a natural language goal, and Mino navigates the page, clicks elements, fills forms, and extracts data - just like a human would.
