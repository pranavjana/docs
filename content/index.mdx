# Mino Intern Docs

Welcome to the Mino Intern Documentation site. This is your central resource for building use cases with Mino's AI-powered web automation API.

## What is Mino?

Mino is an **AI-powered web automation API** by Tinyfish.ai that lets you automate any website using natural language. Instead of writing CSS selectors or XPath, you describe tasks in plain English.

## API at a Glance

```bash
# Simple extraction example
curl -N -X POST https://mino.ai/v1/automation/run-sse \
  -H "X-API-Key: $MINO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://scrapeme.live/shop",
    "goal": "Extract first 3 product names and prices"
  }'
```
